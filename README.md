# 🌾 Concurso Agrinho 2026 - Categoria Programação

## 📋 Informações do Projeto
- **Subcategoria:** 3 - Programação Front-End (HTML, CSS e JavaScript)
- **Tema:** "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"
- **Objetivo:** Jogo Interativo da Memória focado em práticas agrícolas sustentáveis e preservação ecológica.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica de formulários, botões de ação e seções de visualização.
- **CSS3:** Estilização baseada em variáveis nativas para suporte a Temas Claro e Escuro, Layout Flexbox e CSS Grid para construção do tabuleiro de forma responsiva.
- **JavaScript (Vanilla):** Lógica funcional de embaralhamento aleatório, captura de eventos no DOM e controle de acessibilidade textual.

## 🕹️ Instruções de Uso
1. Insira o seu nome na tela inicial do site e clique em **Entrar e Jogar** para liberar o acesso ao tabuleiro.
2. O jogo exibirá uma coleção de cartas ocultas com conceitos fundamentais de equilíbrio sustentável.
3. Clique em duas cartas por rodada para tentar encontrar os pares corretos de tecnologia ecológica (exemplo: Plantio Direto, Energia Solar, Irrigação Inteligente).
4. O placar atualiza em tempo real a quantidade de acertos e tentativas necessárias até que ocorra o encerramento do jogo e a mensagem de vitória seja exibida.
5. Use os botões no topo superior direito para ativar o **Modo Escuro** ou aumentar/diminuir o tamanho das fontes dinamicamente.
